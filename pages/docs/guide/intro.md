# Introduction

Welcome to the wiki guide. This page lives at `docs/guide/intro` — the **second level** of the directory tree.

## Quick start

1. Sign in with your credentials.
2. Navigate to any page using the URL or the **Index** panel.
3. Admin users can edit pages with the **Edit** button or create new ones by navigating to a URL that does not exist yet.

## Page URLs

URLs map directly to the file path under `pages/`, without the `.md` extension:

| File | URL |
|------|-----|
| `pages/docs/guide/intro.md` | `docs/guide/intro` |
| `pages/docs/guide/advanced/tips.md` | `docs/guide/advanced/tips` |

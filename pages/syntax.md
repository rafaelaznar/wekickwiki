# Markdown Syntax

## Headings

```
# H1
## H2
### H3
```

## Emphasis

```
**bold**   *italic*   ~~strikethrough~~
```

## Lists

```
- Item
- Another
  - Sub-item

1. First
2. Second
```

## Links

```
[Label](page)                  ← internal relative link
[External](https://example.com)
```

Relative links are resolved from the current page's path. Use `../` to go up a level.

## Code

```
`inline code`

~~~
code block
~~~
```

## Blockquotes

```
> A block quote
```

## Tables

```
| Column 1 | Column 2 |
|----------|----------|
| cell     | cell     |
```

## Images

```
![alt text](path/image.png)
```

## Horizontal rule

```
---
```
